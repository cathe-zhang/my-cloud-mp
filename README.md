小程序能做和不能做的：
1. 便于传播，用完即走（小工具），而且有内存和包大小的限制，所以不能做太复杂的应用
2. 聊天功能很容易被微信干掉 
3. 个人用户不能做区块链、搜索、新闻等功能
4. 小程序不能主动给用户发消息或通知（公众号可以）
  - 只有在用户提交表单或用户支付后一周内可以发一条消息
5. 腾讯不让做的，你就不能做
